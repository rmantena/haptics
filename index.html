Time-Delayed Bilateral Teleoperation using Wave Variables

We worked on this project to understand how using Wave Variables for Time Delayed Teleoperation would ensure Stability.

Hardware used for this project : 2 Novint Falcons.

SDK used for this project : Force Dimension SDK 3.6 on Windows.

A user selectable (non-varying) Time Delay has been implimented in code. The executable can present time delays between 0.001 sec (actual Haptic loop) and 1 sec.


To run the executable :

  • Download and Install the Falcon Drivers from Force Dimension's website.
  • Download the executable "runMe.exe" and Run it.
  • The executable offers you to experience the following three Bilateral Teleoperation Control Strategies : Bilateral Teleoperation (Position Servo) without time delay, Bilateral Teleoperation with time delay and Bilateral Teleoperation using Wave Variables with time delay.
  • The user should be able to navigate between the three Control Strategies by keyboard inputs. Gains and Time Delays can also be varied.

To modify the source code :

  • Download and Install the SDK from Force Dimension's website.
  • Update the mirror.cpp file at ...\sdk-3.6.0\examples\CLI\mirror\ with our sourceCode.cpp.
  • Compile and Run the updated mirror project.
  • More instructions available in readMe.txt.

References :
